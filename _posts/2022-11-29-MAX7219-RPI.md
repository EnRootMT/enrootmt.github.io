# Introduction

In this article, I will explain how to connect a **MAX7219 LED matrix** to a **Raspberry Pi**.

You will need:
- A Raspberry Pi  
- Linux installed (Raspberry Pi OS recommended)  
- A MAX7219 LED matrix module

# Libraries and modules installation

In a terminal, type the following commands:

~~~bash
sudo usermod -a -G spi,gpio pi
sudo apt-get update
sudo apt-get install build-essential python3-dev python3-pip libfreetype6-dev libjpeg-dev
sudo -i pip install --upgrade pip setuptools
sudo -H pip install --upgrade luma.led_matrix
~~~

---

# Script

Go to the official repository:

ðŸ‘‰ https://github.com/rm-hull/luma.led_matrix

Download the file **matrix_demo.py** located in the `examples` directory.

Then, in a terminal, go to the directory containing the script and run:

~~~bash
python matrix_demo.py
~~~

You should now see the demo displayed on the LED matrix.

From here, you can modify the script to keep only what you need and fully customize the display.
