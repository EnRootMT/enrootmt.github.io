# Prérequis

* Avoir installé zsh : https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH

# Configuration sur VSCode

* Ouvrir **VS Code**
* Appuyer sur **`Ctrl + Shift + P`**    
* Taper :  
    **`Terminal: Select Default Profile`**
* Choisir **zsh**
* Ouvrir un nouveau terminal.
